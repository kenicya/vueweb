##使用vue项目过程##
 
 1. 再github仓库拉取源码 https://github.com/kenicya/vueweb
 2. 进入源码目录
```javascript
# 安装依赖包
npm install

# 再浏览器打开项目，地址为 localhost:8080
npm run dev

# build for production 
npm run build

```

3. 项目跑起来就可以开始编码啦
